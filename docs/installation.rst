============
Instalação
============


Para instalar o Clean Architecture Python Framework Basic Classes, execute os seguintes comandos no sue terminal:

.. code-block:: console

    $ make install


Uma vez instalado, instale o pacote com:

.. code-block:: console

    $ python setup.py install
